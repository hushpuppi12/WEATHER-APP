## Description
This is a simple Weather App built with **HTML, CSS, and JavaScript**.  
It uses the **OpenWeatherMap API** to show live weather for any city.

## Features
- Search for a city 🌍  
- Shows temperature 🌡  
- Displays weather condition ☁  
- Shows wind speed 💨
